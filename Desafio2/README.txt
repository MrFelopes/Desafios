Para rodar o projeto com a atividade do Desafio 2:

Abrir DesafiosWeb.csproj na pasta API pelo Visual Studio ou outra IDE compatível, e rodar o projeto em localhost. Um console deve aparecer, indicando que a API está funcionando. Após isso, abrir index.html em localhost com Live Server ou similar e consultar CEPs.

ATENÇÃO: Como o projeto está rodando em localhost e não em um servidor na nuvem, as portas locais podem variar de um dispositivo para o outro. Caso o código não rode como esperado, certificar de alterar as portas em app.js e Program.cs!
